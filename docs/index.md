# pybrid-computing

Python library for configuring and controlling anabrid's LUCIDAC/REDAC analog-digital hybrid computers.

## Features

- Async JSONL-based RPC over USB Serial or TCP/IP
- Pydantic data models for the full REDAC entity hierarchy
- High-level `Circuit` API for intuitive analog circuit design
- Multi-device support with proxy mode
- Built-in mock server for development without hardware

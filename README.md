pyanabrid-base-hybrid
=====================

Python package containing the common base abstractions for hybrid functionality,
entities, algorithms and similar constructs.

These are used as bases for the concrete implementations in the computer-specific
packages (e.g. `pyanabrid-redac`) to provide an interface for other shared abstractions.

If you are only writing application code for a single computer, you have no need for
the base classes herein.
